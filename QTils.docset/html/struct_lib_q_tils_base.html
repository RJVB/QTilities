Detailed Description

Convenience facility for obtaining the basic function pointers when loading QTils.dll dynamically, via dlopen() or LoadLibrary(). After obtaining a handle to the library, obtain the address of initDMBaseQTils() and then pass it a pointer to a LibQTilsBase structure. initDMBaseQTils() will copy the function addresses into the respective members of that structure, and returns the size of the LibQTilsBase structure. Used for interfacing from Modula-2. NB!! QTilsM2 (Modula-2) defines several additional members in its interface that are not initialised in C!
