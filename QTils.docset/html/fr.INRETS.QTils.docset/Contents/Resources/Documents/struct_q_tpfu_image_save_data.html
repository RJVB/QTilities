Detailed Description

data structure used by QTpfuSaveImage to cache reusable items. QTpfuSaveImage is a version of the SGI Performer pfuSaveImage function optimised for saving series of screendumps e.g. during a simulation. It reuses buffers allocated in preflight instead of at each invocation. It also generates a QI2M file that can be read by QuickTime if the QTImage2Mov importer is installed. Initialised by init_QTpfuImageSaveData, released by free_QTpfuImageSaveData.
